# Luxury Interactive Developer Portfolio

An Awwwards-grade developer portfolio inspired by the minimal, high-tech aesthetics of Apple, Stripe, Framer, Tesla, and OpenAI.

## Design Highlights
- **Living Synapse Canvas**: A 60fps interactive neural network background responsive to mouse proximity.
- **Glassmorphic Bento Grid**: Saturated blur surfaces with Stripe-style hover gradient borders tracking the cursor.
- **HUD Diagnostics Console**: Realtime latency oscilloscope charts and FPS telemetry metrics.
- **Dynamic Simulated Dashboards**: Real-time canvas/DOM simulations of fintech charts, autopilot LIDAR radar systems, synaptic activations, and threat maps.
- **Cognitive Packet Transmission**: Submission effects emitting localized particle bursts to simulate high-tech security logs.

---

## Technical Stack
- **Structure**: Semantic HTML5 markup
- **Styling**: Vanilla CSS3 Custom Properties (CSS variables), HSL profiles, GPU-accelerated transition keyframes.
- **Logics**: Modern JS ES6 DOM listeners, Canvas 2D engines, and Spring animations.

## Interactive Bento Terminal Directives
Input these directives in the simulated console card in the About section:
- `/bio` - Prints biography metadata details.
- `/specs` - Details dev system setups and hardware telemetry.
- `/projects` - Outlines sandbox project directories.
- `/secret` - Reveals hidden Easter Egg.
- `/clear` - Resets prints.
